Class selector for specific elements

This paragraph and the above title belong to "class_example_2" class and are modified by a class selector. The class selector makes text color violet. Also there is a class selector for 'h2' titles which aligns text to the center.

Multiple class selectors

This paragraph and the above title belong to "class_example_2" and "class_example_3" classes. There are CSS selectors for both classes. This paragraph should appear violet (class_example_2) and in Helvetica font (class_example_3). The title should appear centered (class_example_2 for 'h2' elements), violet (class_example_2) and in Helvetica font (class_example_3).
